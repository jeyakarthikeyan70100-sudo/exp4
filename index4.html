What is a Minimum Spanning Tree?

A Minimum Spanning Tree is a subset of edges that connects all vertices of a weighted, connected and undirected graph with minimum possible total edge weight and without forming cycles.

Connected Graph

Every vertex in the graph must be reachable from every other vertex through a path.

No Cycles

The selected edges must not create a cycle. An MST with V vertices always contains V - 1 edges.

Minimum Cost

The total weight of all selected edges must be as small as possible.

Kruskal's Algorithm

Kruskal's algorithm is a greedy algorithm that builds the MST by selecting the smallest available edge while avoiding cycles.

How Kruskal Works

1 Sort all graph edges in increasing order of weight.
2 Select the smallest edge.
3 Check whether adding the edge creates a cycle.
4 If no cycle is created, add the edge to MST.
5 Repeat until V - 1 edges are selected.

Advantages

  • Easy to implement
  • Works efficiently with sparse graphs
  • Uses Union-Find / Disjoint Set
  • Greedy approach

Prim's Algorithm

Prim's algorithm is a greedy algorithm that starts from one vertex and repeatedly adds the minimum-weight edge connecting the current MST to an unvisited vertex.

How Prim Works

1 Select any starting vertex.
2 Mark the starting vertex as visited.
3 Find the minimum-weight edge connecting visited and unvisited vertices.
4 Add the selected edge and new vertex to MST.
5 Repeat until all vertices are included.

Advantages

  • Efficient for dense graphs
  • Works well with priority queues
  • Starts from any vertex
  • Simple graph traversal approach

Algorithm Implementation

Basic Python implementation of both MST algorithms.

Kruskal's Algorithm - Python 

def kruskal(graph, vertices):

    parent = {v: v for v in vertices}

    def find(v):
        if parent[v] != v:
            parent[v] = find(parent[v])
        return parent[v]

    def union(a, b):
        root_a = find(a)
        root_b = find(b)

        if root_a != root_b:
            parent[root_b] = root_a
            return True

        return False

    mst = []

    graph.sort(key=lambda edge: edge[2])

    for u, v, weight in graph:

        if union(u, v):
            mst.append((u, v, weight))

        if len(mst) == len(vertices) - 1:
            break

    return mst

Prim's Algorithm - Python 

import heapq

def prim(graph, start):

    visited = set()
    mst = []
    pq = [(0, start, None)]

    while pq:

        weight, vertex, parent = heapq.heappop(pq)

        if vertex in visited:
            continue

        visited.add(vertex)

        if parent is not None:
            mst.append((parent, vertex, weight))

        for neighbor, edge_weight in graph[vertex]:

            if neighbor not in visited:
                heapq.heappush(
                    pq,
                    (edge_weight, neighbor, vertex)
                )

    return mst

Conclusion

Kruskal's and Prim's algorithms are fundamental greedy algorithms used to construct Minimum Spanning Trees. Kruskal's algorithm processes edges in increasing order and uses the Union-Find data structure to prevent cycles. Prim's algorithm grows the MST from a selected starting vertex by repeatedly choosing the minimum weight edge connecting the current tree to a new vertex. Both algorithms produce an MST for a connected, weighted, undirected graph and are important techniques in graph theory, network design and optimization.
